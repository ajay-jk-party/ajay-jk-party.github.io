Gifts

Gifts and donations

Thank you for thinking about it. Some social trads and cultures make sense and it is justified, however, we both feel that the idea of gifting in events like this is unnecessary, unfortunately, we are more and more getting materialistic. So we request you to not bring any gifts, if you would like to gift, please donate to the following open-source software (that we have heavily abused and continue to do so and procrastinate donating) and/or some of these really cool non-profits. There is some happiness in returning back, so if you would like to gift, please give here!

Non-Profits

  • 80,000 Hours - An organization that helps people have high-impact careers working on the world's most pressing problems, with a focus on AI safety and existential risks.

Open Source Software

  • Sublime Text - A powerful text editor that has been our go-to tool for coding and writing.
  • Python Software Foundation - The foundation behind Python, one of the most widely used programming languages that powers much of modern software.
  • Linux Foundation - Supports the development of Linux and open source technologies that form the backbone of countless systems worldwide.
  • Git - The version control system that has revolutionized software development and collaboration.

Vipassana

  • Bhumi Dhamma - Support for Vipassana meditation centers that provide free meditation courses and teachings.

Apps

  • Signal - A privacy-focused messaging app that we use daily for secure communication.

Education

  • PS Senior School - Support for JK's alma mater, contributing to education and future generations.
